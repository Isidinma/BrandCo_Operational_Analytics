## Data
The operational dataset used in this analysis is proprietary to BrandCo Nigeria Limited. 
Customer names have been anonymised in accordance with data privacy requirements.

## Techniques Applied
1. Classification Model (Random Forest & Logistic Regression)
2. Model Evaluation & Explainability (SHAP)
3. Customer/Order Segmentation (K-Means Clustering)
4. Dimensionality Reduction (PCA)
5. Time Series Analysis & Forecasting (ETS)
